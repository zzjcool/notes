Vue 加载文件为文本格式

使用Vue开发的时候，我们可能需要读取文本文件，这个时候需要使用raw-loader

安装

npm install raw-loader --save-dev

使用

这里以读取yaml文件为例：

const foo = require('!raw-loader!pages/template/foo.yaml')
console.log(foo.default)
